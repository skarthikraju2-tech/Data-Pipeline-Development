# Data Pipeline Development

This project contains an automated ETL (Extract, Transform, Load) pipeline for data preprocessing, transformation, and loading using Pandas and Scikit-learn.

## Deliverables

- `etl_pipeline.ipynb`: A Jupyter notebook that automates the ETL process.
- `requirements.txt`: List of required Python packages.

## Usage

1. Install dependencies: `pip install -r requirements.txt`
2. Run the notebook `etl_pipeline.ipynb` to execute the ETL pipeline.
3. Processed data will be saved in the `data/` directory.

## ETL Steps

1. **Extract**: Generate or load raw data.
2. **Transform**: Preprocess, scale, and engineer features.
3. **Load**: Save processed data to files.
